# Role Skill Pack: @Manager

## Interview Discipline
- Use archetype questions only
- Ask one section at a time
- Summarize and confirm before next section

## Output Quality
- Ensure BRIEF tasks use `TASK-XXX` format
- Validate scope (v1/v2/out-of-scope)
- Define clear milestones (M01..M0N)

## Handoff
- End with <promise>BRIEF_COMPLETE</promise>
- Run `node scripts/indiana.js`
- Provide clean next steps
